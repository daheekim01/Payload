## 🪅 핸들러 (Handler)

* **핸들러**는 **일반적인 용어**로, 특정 이벤트나 동작을 처리하는 **함수**나 **코드**를 의미합니다.
* 이벤트뿐만 아니라 **기타 종류의 동작**도 핸들러를 통해 처리할 수 있습니다.

  * 예를 들어, 버튼 클릭, 서버 응답 처리, 오류 처리 등을 담당하는 함수가 핸들러가 될 수 있습니다.
  * **이벤트 핸들러**도 핸들러의 한 종류이지만, 핸들러는 더 넓은 범위에서 사용됩니다.

* **버튼 클릭 핸들러**: 버튼을 클릭했을 때 특정 작업을 처리하는 함수.
* **서버 응답 핸들러**: 서버에서 데이터를 받아왔을 때 그 데이터를 처리하는 함수.


| **구분**          | **핸들러**                                   | **이벤트 핸들러**                              |
|-------------------|--------------------------------------------|--------------------------------------------|
| **정의**          | 일반적인 동작을 처리하는 함수나 메서드       | 특정 이벤트가 발생할 때 실행되는 함수      |
| **용도**          | 이벤트뿐만 아니라 다양한 작업을 처리        | 이벤트 처리에만 사용                        |
| **예시**          | 서버 응답 처리, 오류 처리                   | 클릭, 키 입력, 마우스 이벤트 처리           |
| **범위**          | 넓은 범위 (이벤트 외의 처리도 포함)         | 이벤트에 국한됨                             |

- **핸들러**: 특정 동작을 처리하는 함수나 메서드.
- **이벤트 핸들러**: 이벤트가 발생할 때 그 이벤트를 처리하는 함수.

---

| **항목**       | **설명**                                       |
| ------------ | -------------------------------------------- |
| `layer`      | Netscape에서 사용되던 레이어 태그, 현재는 HTML5에서 지원되지 않음  |
| `applet`     | Java 애플릿을 삽입하는 태그, 보안 문제로 더 이상 사용되지 않음       |
| `document`   | HTML 문서를 구성하는 DOM 객체                         |
| `xml`        | XML 데이터 처리에 사용되는 포맷, XSS 공격 발생 가능            |
| `link`       | 외부 리소스(CSS 등)를 연결하는 태그                       |
| `msgbox`     | 메시지 박스를 표시하는 함수                              |
| `object`     | 다양한 외부 리소스를 삽입하는 HTML 태그                     |
| `embed`      | 외부 콘텐츠를 삽입하는 HTML5 태그                        |
| `frame`      | 여러 콘텐츠 영역을 생성하는 태그, `<iframe>`으로 대체됨         |
| `base`       | 문서 내에서 상대적인 URL 경로를 설정하는 태그                  |
| `bgsound`    | Internet Explorer에서 배경 음악을 재생하기 위한 태그        |
| `binding`    | XML과 XSLT에서 데이터와 UI 바인딩에 사용되는 속성             |
| `vbscript`   | Microsoft의 VB 기반 스크립팅 언어, 보안상 대부분 지원되지 않음    |
| `charset`    | 문자 인코딩을 설정하는 `<meta>` 태그 속성                  |
| `expression` | IE에서 사용된 CSS 속성, 보안에 취약할 수 있음                |
| `string`     | JavaScript 객체로, 문자열을 다룰 때 사용되며 XSS 공격 가능성 있음 |
| `append`     | DOM에 노드를 추가하는 메서드                            |
| `meta`       | 문서의 메타데이터를 설정하는 HTML 태그                      |
| `prompt`     | 사용자 입력을 받는 대화 상자                             |
| `confirm`    | 사용자 확인을 받는 대화 상자                             |
| `title`      | 문서 제목을 설정하는 HTML 태그                          |
| `void`       | JavaScript에서 표현식의 반환값을 무효화하는 연산자             |
| `refresh`    | 페이지를 자동으로 새로 고침하는 `<meta>` 태그 속성             |
| `blink`      | 텍스트를 깜박이게 하는 HTML 태그, 현재는 대부분 지원되지 않음        |
| `style`      | 페이지의 스타일을 정의하는 `<style>` 태그                  |
